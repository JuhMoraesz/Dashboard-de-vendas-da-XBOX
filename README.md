# Dashboard-de-vendas-da-XBOX
Analise dos Dados ficticios fornecidos pela DIO com o objetivo de criação de Dashboard como respostas de negocio.
As perguntas foram :
* Qual é o total de ganhos feitos atraves do EA play season pass?
* Qual é o total de ganhos feitos atraves do Minecraft season pass ?
* Qual é o total de ganhos do XBOX game Pass? Quanto dessa quantia tem a renovação automatica ?
* Qual é o total de ganhos feitos por tipo de assinatura?
Para as seguintes perguntas foram realizadas analise do banco de dados, posteriormente foi feito o Dashboard.
